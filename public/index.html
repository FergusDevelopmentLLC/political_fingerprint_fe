The Test

PolitiPoint is a test that provides insight into political leanings beyond the traditional "right" and "left" framework.

The test will present you with a series of statements. Indicate from Strongly Agree to Strongly Disagree how closely you identify with the statement.

Your responses will indicate where you place on a continuum between two opposing perspectives in each of four categories (Economic, Diplomatic, Civil and Societal).

Feedback

An important goal of this project is to improve the test in the following ways.

  1. Eliminate statement bias
  2. Simplify the statements
  3. Make the test shorter

By collecting feedback for each statement, subsequent versions of the test can be improved.

Closest match

In addition to the test results, the application will match responses most closely with one of following ideologies. This part of the application is experimental and is a work in progress.

Currently, the logic behind ideology matching is the same as what can be found at 8values.

What data is being collected?
  1. Feedback for each question
  2. Test results by U.S. county

Client IP address is used to approximate county location. Test takers can also select their county approximation, and choose not to aggregate their results with others on the map.
